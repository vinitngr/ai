# Quick Setup

### 1. Global Install
```bash
npm install -g @vinitngr/serper-v
```

### 2. Set API Key
```bash
serperV auth YOUR_API_KEY_HERE
```

### 3. Verify
```bash
serperV search -q "Latest new about openClaw"
```

---
**Note:** Built for OpenClaw agents. Supports bulk scraping and date-range filters.
